A better framework for easily building better apps, which bases on Koa.

## Installation

```bash
npm install --save salak@next
```

Node.js > 7.6.0 required.

## Features

- ✔︎ Full validations
- ✔︎ Auto generate docs
- ✔︎ Auto bind routes
- ✔︎ Plugin system
- ✔︎ Easy extends
